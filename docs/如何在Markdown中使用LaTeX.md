## 字体样式

- 换行：`\\`
- 空格：`\`
- 居中：使用 `$$` 包裹
- 字体样式：加粗：`\mathbf`、斜体：`\mathit`
- 字体大小：`\tiny`、`\scriptsize`、`\small`、`\normalsize`、`\large`、`\Large`、`\huge`、`\Huge`
- 颜色：`\color{颜色值}{数/字/符}`
- 注释：`\text{内容}`

## 上下标

- 上标：`^`
- 下标：`_`
- 如有嵌套，使用 `{}` 包含。

例： $ X^{2m}_{3n} $
