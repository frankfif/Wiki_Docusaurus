## Win+R 常用命令

- control 控制面板
- regedit 注册表编辑器
- eventvwr 事件查看器
- mmc Microsoft 管理控制台
- gpedit.msc 组策略
- devmgmt.msc 设备管理器
- diskmgmt.msc 磁盘管理
- services.msc 服务
- certmgr.msc 证书
- taskschd.msc 计划任务程序
- wf.msc 高级安全 Windows 防火墙
