## 对比

|                | 推挽输出 | 开漏输出                               |
| -------------- | -------- | -------------------------------------- |
| 高电平驱动能力 | 高       | 看外部上拉电阻                         |
| 低电平驱动能力 | 高       | 高                                     |
| 电平跳变速度   | 快       | 由外部上拉电阻决定，越小越快但功耗越大 |
| 线与           | 不支持   | 支持                                   |
| 电平转换       | 不支持   | 支持                                   |
