绘制原理图使用 OrCAD Capture CIS（开始菜单-》Cadence-》Capture CIS）  
绘制 PCB 使用 Allegro PCB Designer （开始菜单-》Cadence-》PCB Editor）

http://cip-server.local/CIP-E/

操作指南：

- 拉线：`W`
- 取消：`ESC`
- 拉排线：`F4`
- 加标签：`N`
